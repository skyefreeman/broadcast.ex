      Broadcast

      Broadcast to multiple social media sources at once.

      Currently supports write-only posting to:

      • Bluesky
      • Mastodon

      Installation

      If available in Hex, the package can be installed by adding broadcast to your list of dependencies in mix.exs:

      def deps do
  [
    {:broadcast, "~> 0.1.0"}
  ]
end
